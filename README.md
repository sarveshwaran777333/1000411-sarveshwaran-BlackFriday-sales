# InsightMart: Black Friday Data Mining

## Project Description:
InsightMart is a web-based advanced analytics dashboard developed using Python, Streamlit, and Machine Learning algorithms. The application allows users to explore Black Friday sales data, segment customers, discover product associations, and isolate spending anomalies. The web app is designed for business analysts and marketing teams, presenting complex data in an easy-to-understand visual format to drive strategic decisions.

## Purpose of the Project
The purpose of InsightMart is to:
1. Unlock business intelligence from historical Black Friday sales data.
2. Segment customers using machine learning to identify valuable demographics.
3. Discover product associations to inform merchandising and bundle discounts.
4. Provide actionable, data-driven marketing strategies based on the findings.

## Main Features:
1. **Executive Command:** Real-time macroeconomic analysis and demographic tracking metrics.
2. **Dimensional Clustering:** 3D interactive visualizations of customer segments (e.g., "High-Spend Veterans" vs. "Budget Youth") using K-Means and PCA.
3. **Neural Association Web:** Directed mathematical graphs showing product categories that are frequently bought together using the Apriori algorithm.
4. **Outlier Isolation:** Multi-algorithmic detection (Isolation Forest, Z-Score) identifying anomalous, high-spending transactions.
5. **Strategic Consulting:** Clear, actionable business recommendations based on the data patterns discovered.

## How the Web App Works:
1. The user opens the InsightMart web application.
2. The user navigates through the sidebar menu to select different analytical modules (e.g., Dimensional Clustering, Neural Association Web).
3. The app processes the underlying Black Friday dataset using embedded machine learning models.
4. The results are generated and displayed instantly through interactive charts and dataframes.
5. The user can review the "Data Architecture" section to read the final strategic consulting advice.

## Technologies Used:
1. **Streamlit** – for building the interactive web interface
2. **Python** – backend logic and mathematical preprocessing
3. **Scikit-Learn & MLxtend** – machine learning models (Clustering, Isolation Forest, Apriori)
4. **Plotly & NetworkX** – interactive 3D graphs and network topography
5. **Pandas & NumPy** – data cleaning and numerical manipulation

## Target Users:
1. Business Analysts
2. Marketing Strategists
3. Retail Store Managers
4. Data Science / IT Students 

## Limitations:
1. Requires an active internet connection to access the Streamlit Cloud deployment.
2. The analysis is currently limited to the static "Black Friday" dataset provided for this assignment.
3. Depending on the device, generating the 3D PCA clusters and Association networks may take a few seconds to process.
